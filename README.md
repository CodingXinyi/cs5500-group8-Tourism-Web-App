# CS5500-Group8-Tourism-Web-App
## Run the project
### Start the react app
Open a Terminal 
```cd frontend```
```npm start```

### Start the backend server
Open a new Terminal 
```cd backend```
```npx nodemon server/api.js```

### Start the prisma studio
Open a new Terminal
```cd backend```
```npx prisma studio --schema=server/prisma/schema.prisma```
